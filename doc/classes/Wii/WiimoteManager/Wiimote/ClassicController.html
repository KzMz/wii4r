Class Wii::WiimoteManager::Wiimote::ClassicController
In: ext/wii4r.c
Parent: Object

Provides a set of methods for accessing ClassicController properties and members. 

     - button events (pressed, released, held, just pressed)
     - joystick values (angles & magnitude)

Public Instance methods

classicctrl.held?(button) → true of false

Returns true if button is being held on self.

classicctrl.just_pressed?(button) → true of false

Returns true if button is just pressed on self.

classicctrl.left_joystick_angle → float

Returns the angle at which the left joystick is being held. 0 => Straight up 90 => right 180 => down 270 => left

classicctrl.left_joystick_magnitude → float

Returns the magnitude at which the left joystick is being held. Range: [0, 1] 0 => center 1 => edges

classicctrl.left_shoulder → float

Returns the magnitude at which the left shoulder button is being held. Range: [0, 1] 0 => not pressed 1 => fully pressed

classicctrl.pressed?(button) → true or false

Returns true if button is being pressed on self.

classicctrl.released?(button) → true or false

Returns true if button is just released on self.

Returns the magnitude at which the right joystick is being held. Range: [0, 1] 0 => center 1 => edges

classicctrl.right_shoulder → float

Returns the magnitude at which the right shoulder button is being held. Range: [0, 1] 0 => not pressed 1 => fully pressed
