Class Wii::WiimoteManager::Wiimote
In: wii4r.c
Parent: Object

Provides a set of methods to access Wiimote functionalities. A Wiimote object can:

  • put the wiimote in rumble state
  • control the leds
  • control and access accelerometer measurement
  • control and access ir camera measurement
  • manage expansions
  • manage pressed, held, released buttons
  • access battery level, acceleration vector and gravity vector

wiimote.absolute_pitch → float

Returns the absolute rotation around X axis (works with motion sensing activated). Range: [-179.9, -0.1] [0.1, 179.9]

wiimote.absolute_position → array

Returns an array containing the absolute position [x, y] of self.

wiimote.absolute_roll → float

Returns the absolute rotation around Y axis (works with motion sensing activated). Range: [-179.9, -0.1] [0.1, 179.9]

wiimote.accel_threshold → int

Returns the accelerometer event threshold for self.

wiimote.accel_threshold = t → int

Sets the accelerometer event threshold for self.

wiimote.acceleration → array

Returns an array containing the three components of acceleration [x, y, z] of self.

wiimote.aspect_ratio → string

Returns a string identificating the aspect ratio of the TV/monitor.

wiimote.aspect_ratio = ratio → (value of aspect ratio constant)

Sets the aspect ratio of the TV/monitor of self to ratio.

wiimote.battery_level → float

Returns the battery level of self.

wiimote.connected? → true or false

Returns true if self is connected via bluetooth, false otherwise.

wiimote.distance → float

Returns the distance of self from the ir sources (works only with ir tracking enabled and at least two ir source visible).

wiimote.expansion? → true or false
wiimote.expansion?(exp) → true or false

The first form returns true if a expansion is inserted into self. The second one return true if the expansion exp is inserted into self.

Returns an array containing the three components of gravity force [x, y, z] of self.

wiimote.has_classic_controller? →true or false

Returns true if a classic controller is attached to self.

wiimote.has_guitar_hero_controller? → true or false

Returns true if a guitar hero 3 controller is attached to self.

wiimote.has_nunchuk? → true or false

Returns true if a nunchuk is attached to self.

wiimote.held?(button) → true of false

Returns true if button is held on self.

wiimote.just_pressed?(button) → true of false

Returns a string which represents the led actually on.

Turns on the led or the leds contained in led.

    wmote.leds = LED_1      #=>(LED_1 value)

    #turns on led 2 and 3
    wmote.leds = LED_2 | LED_3 #=>(LED_2 | LED_3 value)

Sets the motion sensing of self to true or false.

wiimote.motion_sensing? → true or false

Returns true if motion sensing is activated in self, false otherwise.

Returns the rotation around X axis (works with motion sensing activated). Range: [-179.9, -0.1] [0.1, 179.9]

wiimote.position → array

wiimote.rumble! → nil
wiimote.rumble!(duration) → nil
wiimote.rumble!(duration, times) → nil

The first form puts self in rumbling state. The second one puts self in rumbling state for duration seconds. The last one puts self in rumbling state for duration seconds for times times, separated by 1 sec of pause.

wiimote.rumble = true or false → nil

Set the rumble property of self to true or false and makes the device rumble or stop.

wiimote.rumble? → true or false

Returns true if the self is rumbling.

wiimote.sensitivity → int

Returns the sensitivity level of ir camera. Range: [1, 5]

wiimote.sensitivity = level → int

Sets the sensitivity level of ir camera to level. The value will be truncated to 5 or 1.

wiimote.sensor_bar_position → string

Returns a string which represents the position of the sensor bar relative to self.

wiimote.sensor_bar_position = pos → int

Sets the sensor bar position relative to self to pos.

wiimote.speaker? → true or false

wiimote.status → hash

Returns a hash containing the status information of self. 

    h = wmote.status

    h[:ir]              #=> true or false
    h[:attachment] #=> string defining the expansion inserted
    h[:id]              #=> int defining the id of <i>self</i>
    h[:battery]    #=> float (battery level)
    h[:led]             #=> string (the last led turned on)
